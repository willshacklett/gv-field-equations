# FLRW Embedding

## Goal
Define how GV contributes to expansion.

Start with:
\[
H^2 = \frac{8\pi G}{3}\rho_{\text{eff}}(GV)
\]

## Mapping (v0)
\[
\rho_{\text{eff}}(GV)=\rho_m + \rho_r + \rho_{GV}
\]

with (placeholder):
\[
\rho_{GV}=\frac{1}{2}K(GV)\dot{GV}^2 + V(GV)
\]

and pressure:
\[
p_{GV}=\frac{1}{2}K(GV)\dot{GV}^2 - V(GV)
\]

So:
\[
w_{GV}=\frac{p_{GV}}{\rho_{GV}}
\]
